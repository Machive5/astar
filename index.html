HOW TO USE

1. Input how much row and column area you want to create and click generate

2. Check the "Add block" checkbox and click on the cyan square to adding a blocked area. The color will change when you click it.

3. If the "Add block" checkbox is not checked, Then if you click on the cyan square it will add the start and end point. The square color will change to blue if you do it. When two square are choosen the algorithm will running.

4. when it's done you can click on the reset button to reset it or you can click generate to make new area.

5. The delay option is to setting the speed of the visualizer.
